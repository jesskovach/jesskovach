# Security Policy

This repository hosts a personal website and public-facing documentation.

## Supported Versions

This project does not provide versioned software releases or security patches.

## Reporting a Vulnerability

If you believe you have identified a security-related issue affecting this repository or its deployment, please report it by email:

**security@kovachconsultants.com**  

Please do not use public issues to report potential security concerns.

Reports will be reviewed on a best-effort basis.
